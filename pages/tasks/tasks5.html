Задание 5 ОГЭ по информатике. У исполнителя 2 команды. ФИПИ

Задания 5. ОГЭ по информатике необходимо найти:

 - либо алгоритм с помощью данных команд,

 - либо одну из команд с помощью данного алгоритма.

Условии в заданиях такие:

 - Составьте алгоритм получения из числа 89 числа 24, содержащий не более 5 команд. В ответе запишите только номера команд.

 - Известно, что программа 12111 переводит число 7 в число 51. Определите значение b.

Лайфхак:

Если в задании сказано: Определите значение b, то лучше идти по командам которые даны их будет 5 или 4, например, 12111

Если сказано: Составьте алгоритм, то при составлении алгоритма, если в командах есть:

 «команда умножить»,

 «команда возведи в квадрат»,

 «приписать число»

то лучше всего взять обратные команды:

 «извлечь из-под корня»,

 «раздели»,

 «убрать число»

Например:

 1. возведи в квадрат => извлечь из-под корня

 2. умножь на 3 => раздели на 3

 3. приписать 2 => убрать 2 (12 => 1)

 из числа 2 числа 64 => из числа 64 числа 2

1) У исполнителя Квадратор две команды, которым присвоены номера:

1. возведи в квадрат
2. вычти 4

Первая из них возводит число на экране во вторую степень, вторая уменьшает число на 4. Исполнитель работает только с натуральными числами.
Составьте алгоритм получения из числа 2 числа 64, содержащий не более пяти команд. В ответе запишите только номера команд.
(Например, 12221 –
это алгоритм:
возведи в квадрат
вычти 4
вычти 4
вычти 4
возведи в квадрат,
который преобразует число 5 в число 169.)
Если таких алгоритмов более одного, то запишите любой из них.

Решение

Нужно составить алгоритм чтобы получить из числа 2 числа 64.
Смотрим первая команда «возведи в квадрат». Поэтому возьмем обратные команды.

1. возведи в квадрат => 1. извлечь из-под корня

2. вычти 4 => 2. прибавь 4

из числа 2 числа 64 => из числа 64 числа 2

64
извлекается корень, да,

извлечь из-под корня = 8 (1 команда)

8 извлекается корень, нет, поэтому прибавь 4 и т. д.

прибавь 4 = 12 (2 команда)

прибавь 4 = 16 (2 команда)

извлечь из-под корня = 4 (1 команда)

извлечь из-под корня = 2 (1 команда)

2

Из числа 64 получили число 2, в ответе записываем номера команд из условия задачи.
Так как в начале брали обратные команды, то полученный алгоритм пишем обратно, то есть: 12211 => 11221

Ответ: 11221

2) У исполнителя Альфа две команды, которым присвоены номера:

1. прибавь 1
2. умножь на b
(b  — неизвестное натуральное число; b ≥ 2).

Выполняя первую из них, Альфа увеличивает число на экране на 1, а выполняя вторую, умножает это число на b. Программа для исполнителя Альфа  — это последовательность номеров команд.
Известно, что программа 11211 переводит число 6 в число 82. Определите значение b.

Решение

Нужно определите значение b. Из числа 6 получили число 82, с программой 11211.

6
6 + 1 = 7 (1 команда)
7 + 1 = 8 (1 команда)
8 * b = 8b (2 команда)
8b + 1 = 8b + 1 (1 команда)
8b + 1 + 1 = 8b + 2 (1 команда)

8b + 2 = 82
8b = 82 - 2
8b = 80
b = 80 : 8
b = 10

Ответ: 10

3) У исполнителя Конструктор две команды, которым присвоены номера:

1. приписать 2
2. разделить на 2

Первая из них приписывает к числу на экране справа цифру 2, вторая делит его на 2.
Составьте алгоритм получения из числа 14 числа 9, содержащий не более 5 команд.. В ответе запишите только номера команд.
(Например, 22212 –
это алгоритм:
разделить на 2
разделить на 2
разделить на 2
приписать 2
разделить на 2,
который преобразует число 8 в число 6.)
Если таких алгоритмов более одного, то запишите любой из них.

Решение

Нужно составить алгоритм чтобы получить из числа 14 числа 9.
Смотрим первая команда «приписать 2». Поэтому возьмем обратные команды.

1. приписать 2 => 1. убрать 2

2. разделить на 2 => 2. умножь на 2

из числа 14 числа 9 => из числа 9 числа 14

9
можно убрать 2, нет, поэтому умножаем на 2

умножь на 2 = 18 (2 команда)

18 можно убрать 2, нет, поэтому умножаем на 2 и т. д.

умножь на 2 = 36 (2 команда)

умножь на 2 = 72 (2 команда)

убрать 2 = 7 (1 команда)

умножь на 2 = 14 (2 команда)

14

Из числа 9 числа 14, в ответе записываем номера команд из условия задачи.
Так как в начале брали обратные команды, то полученный алгоритм пишем обратно, то есть: 22212 => 21222

Ответ: 21222

4) У исполнителя Гамма две команды, которым присвоены номера:

1. прибавь 3
2. умножь на b
(b  — неизвестное натуральное число; b ≥ 2).

Выполняя первую из них, Гамма увеличивает число на экране на 3, а выполняя вторую, умножает это число на b. Программа для исполнителя Гамма  — это последовательность номеров команд.
Известно, что программа 11211 переводит число 1 в число 97. Определите значение b.

Решение

Нужно определите значение b. Из числа 1 получили число 97, с программой 11211.

1
1 + 3 = 4 (1 команда)
4 + 3 = 7 (1 команда)
7 * b = 7b (2 команда)
7b + 3 = 7b + 3 (1 команда)
7b + 3 + 3 = 7b + 6 (1 команда)

7b + 6 = 97
7b = 97 - 6
7b = 91
b = 91 : 7
b = 13

Ответ: 13
